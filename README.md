🏥 Healthcare Data Analysis Dashboard (Power BI)
📌 Project Overview

This project focuses on analyzing large-scale healthcare diagnostic data to uncover insights related to operational efficiency, turnaround time (TAT), and performance across different regions and departments.

An interactive Power BI dashboard was developed to monitor KPIs and support data-driven decision-making in healthcare operations.

📊 Dataset Details
📁 Records: 469,000+ test entries
🌍 Coverage: Multiple states and cities
🧪 Includes:
Test Name
Department
Processing Centre
Patient Demographics (Age, Gender)
Booking & Delivery Dates
Test Status (Approved, Rejected, Re-run, etc.)
🎯 Objectives
Analyze Turnaround Time (TAT) performance
Track test rejections, re-runs, and outsourcing trends
Identify top-performing states, cities, and departments
Perform time-based and demographic analysis
Build an interactive dashboard for stakeholders
📈 Key Insights
✅ 469K+ tests analyzed
⏱️ 88%+ tests completed within TAT
🏆 Some states achieved ~100% TAT efficiency
🔁 Identified patterns in test re-runs and rejections
🕒 Peak processing observed during specific hourly slots
🧬 Departments like Immunoassays showed higher efficiency compared to others
📊 Dashboard Features
🔹 KPI Cards (Total Tests, % in TAT, Re-run, Hold, Outsourced)
🔹 State-wise & City-wise performance analysis
🔹 Department & Lab efficiency comparison
🔹 Age group analysis
🔹 Hourly trend analysis for sample processing
🔹 Dynamic slicers:
Booking Date
Department
City
Doctor Name
Test Name
🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Modeling
Data Cleaning & Transformation
